## 使用方法
### 基本命令

```bash
# 启动模块
can_test_link start [options]

# 停止模块
can_test_link stop

# 查看状态
can_test_link status
```

### RX 模式（接收）

在飞控 B 上启动接收模式：

```bash
can_test_link start -m rx -d can0
```

日志输出示例：
```

INFO  [can_test_link] Socket created: fd=3
INFO  [can_test_link] Interface index: 1
INFO  [can_test_link] CAN_RAW_LOOPBACK disabled
INFO  [can_test_link] CAN_RAW_RECV_OWN_MSGS disabled
INFO  [can_test_link] Socket bound successfully to can0 (ifindex=1)
INFO  [can_test_link] Module started successfully
INFO  [can_test_link] Poll timeout: no frame received yet (waiting for first frame)
INFO  [can_test_link] RX[1]: ID=0x123 DLC=8 Data=[11 22 33 44 55 66 77 88]
INFO  [can_test_link] RX[2]: ID=0x123 DLC=8 Data=[11 22 33 44 55 66 77 88]
```

### TX 模式（发送）

在飞控 A 上启动发送模式：

```bash
# 默认：ID=0x123, 10Hz
can_test_link start -m tx -d can0

# 自定义 ID 和频率：ID=0x456, 20Hz
can_test_link start -m tx -d can0 -i 0x456 -r 20
```

日志输出示例：
```
INFO  [can_test_link] Starting in TX mode: device=can0, ID=0x123, rate=20 Hz
INFO  [can_test_link] Socket created: fd=3
INFO  [can_test_link] Interface index: 1
INFO  [can_test_link] CAN_RAW_LOOPBACK disabled
INFO  [can_test_link] CAN_RAW_RECV_OWN_MSGS disabled
INFO  [can_test_link] Socket bound successfully to can0 (ifindex=1)
INFO  [can_test_link] Module started successfully
INFO  [can_test_link] TX[1]: ID=0x123 DLC=8 Data=[11 22 33 44 55 66 77 88]
INFO  [can_test_link] TX[2]: ID=0x123 DLC=8 Data=[11 22 33 44 55 66 77 88]
```
